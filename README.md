## Project Overview

This project is a modern web application built with [Next.js](https://nextjs.org), utilizing React 19 and TypeScript for robust type safety and developer experience. It features a component-based architecture, leveraging Tailwind CSS and DaisyUI for rapid UI development and consistent styling.

**Key features:**
- **Product Listing and Details:** Browse products and view detailed information for each item.
- **Sorting and Filtering:** Easily sort products by name or rating, and toggle sort order.
- **Reusable Components:** Includes custom components for modularity and reusability.
- **Optimized Images:** Uses Next.js's `next/image` for fast, responsive image loading.
- **API Integration:** Fetches product data from a backend API for dynamic content.
